# API Methods {#head}

## GET server {#fetch}

Get information about server

### Request parameters {#params}

| Type | Endpoint     | Parameters      | Result                            |
|------|--------------|-----------------|-----------------------------------|
| GET  | /servers/:id | :id - Server ID | [ResourceServer](#resourceserver) |

# Data Types {#types}

## ResourceServer

| Property         | Type                                                 | Description                                    |
|------------------|------------------------------------------------------|------------------------------------------------|
| id               | string                                               | Server ID                                      |
| name             | string                                               | Server name                                    |
| description      | string                                               | Server description (with raw HTML & MD)        |
| shortDescription | string                                               | Short server description (without HTML and MD) |
| invite           | string                                               | Server invite code (discord.gg/HERE)           |
| avatar?          | string                                               | Server icon                                    |
| rating           | number                                               | Average server rating                          |
| memberCount?     | number                                               | Server member count                            |
| standardBannerID | number                                               | Standard banner ID                             |
| bannerURL?       | string                                               | Discord banner link                            |
| tags             | array of [ServerTag](#tags)                          | Server tags                                    |
| shortLink?       | string                                               | Server short link                              |
| inviteLink       | string                                               | Server invite code (deprecated)                |
| owner            | string                                               | Server owner ID                                |
| status           | [ResourceStatus](/api/#ResourceStatus)         | Status in BotiCord search                      |
| ratings          | array of [ResourceRating](/api/ResourceRating) | Server rating                                  |
| website?         | string                                               | Server website                                 |
| moderators       | array of [PartialUser](/api/#PartialUser)      | Server moderators                              |
| upCount          | number                                               | Up Count                                       |
