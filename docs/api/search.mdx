# Search {#head}

To search for bots, servers and reviews, we use the [Meilisearch](https://www.meilisearch.com) service.
You can get acquainted with all the search methods [here](https://www.meilisearch.com/docs/reference/api/overview)

You can get the key for the search using the method `GET /search-key`

# Search indexes {#indexes}

| Name     | Description    | Data                                        |
|----------|----------------|---------------------------------------------|
| bots     | Search bots    | [MeiliIndexedBot](#meiliindexedbot)         |
| servers  | Search servers | [MeiliIndexedServer](#meiliindexedserver)   |
| comments | Search reviews | [MeiliIndexedComment](#meiliindexedcomment) |

# Data types {#types}

## MeiliIndexedComment

| Property | Type   | Description                   |
|----------|--------|-------------------------------|
| id       | string | Review ID                     |
| author   | string | Author ID                     |
| content  | string | Review content                |
| rating   | number | Review rating (from 1 to 5)   |
| resource | string | Bot/Server ID                 |
| created  | string | Date the review was created   |
| modReply | string | Reply to the review from mods |

## MeiliIndexedServer

| Property         | Type                                    | Description                                    |
|------------------|-----------------------------------------|------------------------------------------------|
| id               | string                                  | Server ID                                      |
| name             | string                                  | Server name                                    |
| description      | string                                  | Server description (with raw HTML & MD)        |
| shortDescription | string                                  | Short server description (without HTML and MD) |
| invite           | string                                  | Server invite code (discord.gg/HERE)           |
| avatar?          | string                                  | Server icon                                    |
| ups              | number                                  | Up Count                                       |
| rating           | number                                  | Average server rating                          |
| members?         | number                                  | Server member count                            |
| banner           | number                                  | Standard banner ID                             |
| discordBanner?   | string                                  | Discord banner link                            |
| tags             | array of [ServerTag](/api/servers#tags) | Server tags                                    |
