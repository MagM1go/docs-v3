# API methods {#methods}

## GET user {#get-profile}

Get user profile

### Request parameters {#param-req}

| Type | Endpoint    | Result                      |
|------|-------------|-----------------------------|
| GET  | /users/:id	 | [UserProfile](#UserProfile) |

# Data types {#types}

## UserProfile

| Property          | Type                                          | Description                     |
|-------------------|-----------------------------------------------|---------------------------------|
| username          | string                                        | Username                        |
| discriminator     | string                                        | User discriminator (deprecated) |
| avatar?           | string                                        | User avatar                     |
| id                | string                                        | User ID                         |
| badges            | array of [UserBadge](#UserBadge)              | User badges                     |
| bots              | array of [PartialBot](/api/bots#ResourceBot)  | User bots                       |
| servers           | [PartialServer](/api/servers/#ResourceServer) | User servers                    |
| socials?          | [UserLinkTypes](#UserLinkTypes)               | User social links               |
| description?      | string                                        | Profile description             |
| shortDescription? | string                                        | Short profile description       |

---

## PartialUser

Same as [UserProfile](#UserProfile), but without `badges` `bots` `servers` properties.

---

## UserBadge

Profile badge

| Field	   | Type   | Description    |
|----------|--------|----------------|
| id       | number | Badge ID       |
| name     | string | Badge name     |
| assetURL | string | Badge icon URL |

---

## UserLinkTypes

Links to the user's social networks

| Code | Name     | Description                |
|------|----------|----------------------------|
| 0    | Vk       | vk.com                     |
| 1    | Telegram | t.me                       |
| 2    | Donate   | Link to the donation page  |
| 3    | Git      | Link to the git profile    |
| 4    | Custom   | Link to the custom website |
